### 汇编格式

linux下汇编为AT&T格式，而在windows中则为intel格式，二者区别如下：

|   **规则**   |    **Intel风格**     |     **AT&T风格**     |
| :----------: | :------------------: | :------------------: |
|  操作数顺序  | 目的操作数，源操作数 | 源操作数，目的操作数 |
|    寄存器    |         rax          |         %rax         |
|    立即数    |          1           |          $1          |
|    16进制    |       后缀加H        |       前缀加0x       |
| 访问内存长度 |    前缀BYTE PTR等    |    后缀b,w,q,l等     |
|   绝对寻址   |        [imm]         |         imm          |
|   间接寻址   |        [rap]         |        (%rap)        |
| 基址相对寻址 |      [rap+imm]       |      imm(%rap)       |
|   变址寻址   |      [rbp+rdi]       |     (%rbp+%rdi)      |
| 相对变址寻址 |    [rbp+rdi+imm]     |    imm(%rbp+%rdi)    |
